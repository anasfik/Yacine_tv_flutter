# Creating Admin Credential

## Creation

In order to create a new account which you will let you authenticate to access the Admin Panel, you will need to follow these steps:

1. open the .env file that you've created from the previous section
2. Add your desirer username and password as environment variables, like this :

```
ADMIN_USERNAME : the desired admin username.
ADMIN_PASSWORD : the desired admin password.
```

3. Run the command `npm run create_admin` to create the admin credential using the ADMIN_USERNAME and ADMIN_PASSWORD information you did put inside the .env file.

## Verification

1.  if the operation did executed successfully, your console should print:

```
Admin with username <ADMIN_USERNAME> created successfully
```
